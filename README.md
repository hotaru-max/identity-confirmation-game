① 概要

このシミュレーションゲームは、授業中に起こる児童のさまざまな言動に対して、教師がどのように判断し、対応を選択するのかを体験的に考えるためのものです。
授業の進行や学級の秩序といった制約があるなかで、教師は児童の発言や行動を「問題」として制止するのか、それとも「学びの可能性」として受け止めるのかという判断を迫られます。
本ゲームでは、複数の児童への対応選択を通して、自身の判断傾向や、どのような考えがその判断に影響しているのかを振り返ることができます。
 
 ②やり方

ゲームを開始すると、授業の状況が提示されます。
　学習内容や学級の様子を読み、教師の立場になって状況を把握してください。

授業中の児童の言動が順番に示されます。
　それぞれの児童に対して、いくつかの対応選択肢が表示されます。

最も適切だと思う対応を選択してください。
　正解・不正解はありません。自分が「教師だったらどうするか」を基準に選んでください。

選択後、次の場面へ進みます。
　同様の判断を複数回行い、すべての児童への対応を終えるとシナリオは終了します。

ゲーム終了後、判断に影響した要因について質問があります。
　自身の経験や不安、周囲からの影響など、最も当てはまるものを選んでください。

最後に振り返りを行います。
　自分の判断傾向や、「主体的な児童とは何か」について考えてみてください。

③ 補足

このゲームは教師の考え方を評価するものではありません。

自身の判断のクセや考え方に気づくことを目的としています。

教職志望の学生や、授業づくりについて考えたい方におすすめです。
